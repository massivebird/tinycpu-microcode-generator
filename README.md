# Microcode generator

A TinyCPU microcode generator for COSC 321 (Computer Organization II) :3

## Building

To manually build the project, you must first [install Rust](https://www.rust-lang.org/tools/install).

Once you have Rust installed, run the following commands:

```bash
git clone https://github.com/massivebird/tinycpu-microcode-generator
cd tinycpu-microcode-generator
cargo run # runs unoptimized build
```
